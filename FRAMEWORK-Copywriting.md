# 02 — Copywriting
### Panduan Copywriting untuk Template Presentasi Canva

---

## Alur Kerja Divisi Copywriting

```
TERIMA BRIEF dari Research → BUAT JUDUL → BUAT OUTLINE → TULIS KONTEN → HANDOFF ke Illustration & Layout
```

Divisi ini bertanggung jawab menentukan **judul, struktur, dan isi teks** setiap template.

---

## 1. Format Judul Template

### Formula

```
[Color] + [Style] + [Subject] + [Category]
```

- **Color** dan **Style** → diisi oleh designer (biarkan kosong: ___ ___)
- **Subject** → diisi oleh copywriter
- **Category** → tipe dokumen (Presentation, Instagram Post, dll)

### Contoh

```
___ ___ Thank You Mom Mothers Day Tribute Presentation
___ ___ Mothers Day Employee Appreciation Event Presentation
___ ___ Stories Of Moms In Our Community Mothers Day Presentation
```

### Aturan Penamaan Subject

- Gunakan **variasi keyword** antar judul (classroom, school project, school activity)
- Setiap judul harus mengandung **keyword utama topic** (Mothers Day)
- Sertakan **keyword audience/context** (classroom, office, community)
- Sertakan **keyword tipe konten** (tribute, celebration, event program)

---

## 2. Story Arc — Alur Konten Presentasi

### 5 Fase Universal

Setiap presentasi mengikuti alur 5 fase yang sama, terlepas dari tipe atau audience:

```
┌─────────────────────────────────────────────────────┐
│  OPENING → CORE CONTENT → ENGAGEMENT → PEAK → CLOSE │
│  (1-2 pg)   (4-6 pg)      (1-2 pg)   (1 pg)  (1 pg)│
└─────────────────────────────────────────────────────┘
```

| Fase | Jumlah Halaman | Fungsi |
|---|---|---|
| Opening | 1–2 halaman setelah cover | Sambutan, konteks, kenapa presentasi ini ada |
| Core Content | 4–6 halaman | Isi utama, berbeda tergantung tipe presentasi |
| Engagement Break | 1–2 halaman | Elemen interaktif, fun, memecah ritme |
| Emotional Peak | 1 halaman | Momen paling emosional atau impactful |
| Closing | 1 halaman sebelum back cover | Pesan penutup, call to action, ucapan |

---

## 3. Tipe Halaman

### 7 Tipe Halaman yang Bisa Dikombinasi

| Tipe | Fungsi | Contoh Judul Halaman |
|---|---|---|
| **Opener** | Set the tone, konteks | "Welcome", "This Is For You", "About This Event" |
| **Profile/Spotlight** | Menampilkan orang | "Meet The Moms", "Mom Story 1", "Meet The Committee" |
| **List/Value** | Menyampaikan poin-poin | "What I Love About You", "Lessons You Taught Me" |
| **Quote/Testimony** | Suara orang lain | "In Their Own Words", "What The Community Says" |
| **Interactive/Fun** | Engagement element | "Trivia", "Fun Facts", "Best Dressed Mom" |
| **Emotional/Reflective** | Momen mendalam | "Words I Never Got To Say", "The Ripple Effect" |
| **Practical/Utility** | Informasi konkret | "Event Details", "Schedule", "Menu", "Door Prize" |

### Kombinasi Tipe per Audience

| Tipe Presentasi | Tipe Halaman Dominan |
|---|---|
| Tribute / Personal | Opener, List/Value, Emotional, Quote |
| Corporate / Office | Opener, Profile, Quote, Interactive |
| Community | Opener, Profile, Quote, Emotional |
| Event Rundown | Opener, Practical, Interactive, Profile |
| Educational | Opener, List/Value, Interactive, Emotional |

### Aturan

- Setiap presentasi menggunakan **minimal 4 tipe** dari 7
- **Opener** dan **Emotional/Reflective** wajib ada di setiap presentasi
- **Interactive/Fun** sangat disarankan — meningkatkan engagement dan usage

## 4. Sistem Kolom (Layout Dinamis)

### Kapan Menggunakan Berapa Kolom

| Kolom | Kapan Digunakan | Fungsi |
|---|---|---|
| **1 kolom** | Opening, emotional peak, closing | Ruang untuk teks panjang, font besar, momen emosional |
| **2 kolom** | Perbandingan, profile, story | Kontras dua sisi, foto + teks |
| **3 kolom** | Poin utama, awards, values | 3 gagasan setara |
| **4 kolom** | Detail info, trivia, quotes | Banyak info ringkas sejajar |
| **5 kolom** | Fun facts, keywords, timeline | Single word/angka + ikon |

### Aturan Kolom

- ✅ Halaman pertama setelah cover → **selalu 1 kolom**
- ✅ Halaman terakhir sebelum back cover → **selalu 1 kolom**
- ❌ Tidak boleh ada **2 halaman 1 kolom berturut-turut** di tengah presentasi
- ❌ Tidak boleh ada **2 halaman 5 kolom** dalam satu presentasi
- ✅ Halaman kolom banyak (4–5) **selalu diapit** halaman kolom sedikit (1–2)

### Panjang Teks per Kolom

| Kolom | Panjang Teks per Kolom |
|---|---|
| 1 kolom | 2–3 kalimat |
| 2 kolom | 1–2 kalimat |
| 3 kolom | 1 kalimat, atau heading + 1 kalimat |
| 4 kolom | 1 kalimat pendek |
| 5 kolom | 1–3 kata saja |

**Prinsip:** Semakin banyak kolom → semakin pendek teks. Menjaga keseimbangan visual.

---

## 5. Formula Template Universal

### Untuk Presentasi 12–13 Halaman

```
Page 1  → Cover
Page 2  → Opener (1 kolom) — set the tone
Page 3  → Context/Value (3–4 kolom) — kenapa ini penting
Page 4  → Core Content A (2 kolom) — mulai isi utama
Page 5  → Core Content B (1–2 kolom) — lanjutan atau variasi
Page 6  → Core Content C (3 kolom) — poin pendukung
Page 7  → Core Content D (2 kolom) — perspektif lain
Page 8  → Engagement Break (4–5 kolom) — fun/interaktif/ringan
Page 9  → Social Proof / Quotes (3–4 kolom) — suara orang lain
Page 10 → Emotional Peak (1 kolom) — momen terdalam
Page 11 → Call To Action / Takeaway (3 kolom) — apa yang dilakukan
Page 12 → Closing Message (1 kolom) — penutup emosional
Page 13 → Back Cover
```

### Variasi Jumlah Halaman

| Target | Cara Adjust |
|---|---|
| **10 halaman** | Kurangi Core Content menjadi 2–3 halaman |
| **12 halaman** | Ikuti formula standard di atas |
| **15 halaman** | Tambah Core Content atau Profile/Spotlight 2–3 halaman |

## 6. Panduan Tone & Bahasa

### Tone per Audience

| Audience | Tone | Contoh Kalimat |
|---|---|---|
| Anak/Sekolah | Playful, sederhana | "My mom is special because..." |
| Personal/Keluarga | Emosional, intimate | "You never asked me to be perfect" |
| Corporate | Profesional, warm | "Today we celebrate the incredible moms in our team" |
| Komunitas | Hangat, inklusif | "These are their stories — they deserve to be told" |
| Event Rundown | Informatif, welcoming | "Grab your cup and join us for an afternoon of laughter" |

### Aturan Copywriting

- Gunakan **bahasa Inggris sederhana (Simple English)** agar relevan secara global.
- Hindari teks panjang — patuhi **60% Rule** (sisakan 40% ruang kosong dalam box teks).
- Gunakan **placeholder instruksional** untuk membantu user. Contoh: `[Type your heartfelt message here]` atau `[Insert photo of your favorite memory]`.
- Setiap halaman harus **bisa berdiri sendiri** — user bisa hapus/tambah slide tanpa merusak alur.
- **Strictly No Religious/Political Content**: Hindari referensi agama atau politik kecuali diminta spesifik.
- **No Brand Names**: Jangan menyebut merk (Instagram, Facebook, etc). Gunakan "Social Media" atau "Digital Platform".
- **Active Voice Only**: Gunakan kata kerja aktif seperti *Celebrate, Honor, Share, Join, Create*.

---

## 7. Standar Review Canva (Global Market)

Untuk menjamin template lolos review Canva (Creative Marketplace), patuhi aturan berikut:

| Kategori | Larangan (Strictly Avoid) | Rekomendasi (Always Use) |
|---|---|---|
| **Tone & Style** | Terlalu "Educational", kaku, akademis. | Inspiratif, celebratory, warm, friendly. |
| **Diksi (Academic)** | Lesson Plan, Curriculum, Syllabus, Semester, Module. | Activity, Our Journey, Let's Explore, Group Fun. |
| **Safety & Policy** | Konten harmful, kekerasan, senjata, obat-obatan. | Wholesome, safe for work, inclusive family vibes. |
| **Universal Appeal** | Idiom lokal, slang (gaul), referensi niche. | Bahasa Inggris umum yang mudah dimengerti non-native. |

---

## Deliverable dari Divisi Copywriting

Setelah copywriting selesai, handoff ke tim Illustration & Layout berisi:

1. ✅ Judul template (subject) — sudah final
2. ✅ Outline halaman (judul tiap page)
3. ✅ Jumlah kolom per halaman
4. ✅ Konten teks per kolom per halaman
5. ✅ Catatan khusus untuk designer (placeholder, foto, ikon)
6. ✅ Tone dan style yang direkomendasikan
